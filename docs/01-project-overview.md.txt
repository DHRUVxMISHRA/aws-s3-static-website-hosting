## 💡 What is Amazon S3?

Amazon S3 is an **Object Storage Service**, unlike EBS or EFS which provide block/file storage.

It is designed to store any type of data such as:

- 📄 Documents
- 🖼️ Images
- 🎥 Videos
- 🎵 Audio Files
- 💾 Backups
- 📂 Application Files
- 📦 Logs & Archives

### 🌟 Key Characteristics

| Feature | Description |
| --- | --- |
| 📈 Scalability | Virtually unlimited storage |
| ⚡ Performance | Fast upload & retrieval |
| 🛡️ Security | IAM, Bucket Policies, Encryption |
| 🌍 Availability | 99.99% |
| 💪 Durability | 99.999999999% (11 Nines) |
| 💰 Pricing | Pay only for what you use |

## 📚 Important Terminology

### 🪣 Bucket

A **Bucket** is a container that stores objects.

Think of it like a folder in Google Drive.

```text
Bucket
│
├── image.png
├── notes.pdf
├── video.mp4
└── backup.zip
```

> 💡 **Without a bucket, objects cannot be stored in S3.**

---

### 📄 Object

An **Object** is the actual file stored inside the bucket.

Examples:

- photo.jpg
- movie.mp4
- report.pdf
- index.html

Every object contains:

```text
Object
=
File
+
Metadata
+
Unique Key
```

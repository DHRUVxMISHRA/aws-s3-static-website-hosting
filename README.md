# 🌐 AWS S3 Static Website Hosting Project

[![AWS](https://img.shields.io/badge/AWS-S3-orange?logo=amazonaws)](https://aws.amazon.com/s3/)
[![Status](https://img.shields.io/badge/Project-Completed-brightgreen)]()
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A hands-on AWS project demonstrating how to deploy a **Static Website using Amazon S3** while exploring important S3 concepts such as Versioning, Replication, Storage Classes, Lifecycle Rules, Snow Family, and Storage Gateway.

---

# 📌 Project Overview

This project covers the complete workflow of hosting a static website using Amazon S3.

The project includes:

- 🪣 Creating an S3 Bucket
- 📤 Uploading Website Files
- 🌐 Static Website Hosting
- 🔓 Public Access Configuration
- 📜 Bucket Policy Configuration
- 🔄 S3 Versioning
- 🌍 S3 Replication (SRR & CRR)
- 💾 Storage Classes
- ♻️ Lifecycle Rules
- ❄️ AWS Snow Family
- 🌉 AWS Storage Gateway
